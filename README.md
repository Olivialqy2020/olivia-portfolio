# Olivia Portfolio

A personal portfolio website for Olivia Luo, built with React, Vite, TypeScript, Tailwind CSS, and lucide-react.

## Features

- Responsive one-page portfolio layout
- English / Chinese language toggle
- Work experience, education, projects, and contact sections
- Updated profile content reflecting CPA exams passed and ongoing data/AI coursework

## Local development

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:5173`.

## Production build

```bash
npm run build
npm run preview
```
